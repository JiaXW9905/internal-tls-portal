# Internal TLS Portal 用户操作指南

## 1. 适用对象
本指南面向系统最终用户，包含以下角色的日常操作：
- `service`（申请人）
- `dev`（签发人员）
- `admin`（管理员）
- `product`（总览查看角色）

## 2. 访问入口
- 登录页：`/login`
- 注册页：`/register`
- 找回密码：`/forgot`
- 证书申请：`/`
- 证书签发：`/admin`
- 证书总览：`/overview`
- 用户与权限（管理员）：`/users`
- 系统设置（管理员）：`/settings`
- 账号管理：`/account`

> 页面右下角会显示当前版本号角标，便于核对环境版本。

## 3. 账号相关

### 3.1 注册
1. 打开注册页，填写姓名、公司邮箱、验证码。
2. 点击“发送验证码”。
3. 收到验证码后提交注册。
4. 初始密码为 `123456`，首次登录后建议立即修改。

### 3.2 登录
1. 打开登录页，输入邮箱和密码。
2. 登录后系统将按角色显示可访问菜单。

### 3.3 找回密码
1. 打开找回密码页。
2. 输入邮箱并发送验证码。
3. 填写验证码和新密码后重置。

### 3.4 修改密码
1. 在右上角用户菜单进入“账号管理”。
2. 输入当前密码与新密码，点击保存。

## 4. 申请人（service）操作

### 4.1 新建申请
1. 进入“证书申请”页。
2. 填写客户名称、产品（SKU）、VID、申请类型。
3. 点击“提交申请”。

### 4.2 查看我的申请
1. 在“我的申请”区域点击“加载我的申请”。
2. 查看申请状态：
   - `pending`：待处理
   - `issued`：已签发
   - `withdrawn`：已工单撤回
   - `revoked`：证书已撤销

### 4.3 撤回申请（仅 pending）
1. 找到状态为 `pending` 的记录。
2. 点击“工单撤回”并确认。

### 4.4 下载证书 ZIP
1. 仅 `issued` 且文件可用的记录可下载。
2. 若记录存在但文件缺失，下载按钮会置灰不可点击。
3. 页面会同时显示解压密码（如已配置）。

## 5. 签发人员（dev/admin）操作

### 5.1 处理待处理申请
1. 进入“证书签发”页。
2. 在“待处理申请”中选择工单。
3. 上传 ZIP 文件并填写解压密码。
4. 提交后状态变为 `issued`。

### 5.2 最近已签发列表
- 可查看最近签发记录及证书信息。
- 支持执行“证书撤销”。

### 5.3 证书撤销规则
- 仅存在活跃证书时可撤销。
- 签发超过 7 天后不可撤销。
- 超期记录在操作列显示 `-`（不可操作）。

## 6. 总览查看（admin/dev/product）

### 6.1 证书总览使用方法
1. 进入“证书总览”页。
2. 在“已签发证书”区域可使用：
   - 关键字搜索（客户名/SKU/VID/申请人）
   - 开始/结束日期
   - 即将过期（60 天内）
3. 点击“搜索”更新结果。
4. 点击“导出 CSV”下载当前筛选结果。

### 6.2 过期时间显示规则
- 优先展示证书记录中的显式过期时间；
- 若无显式过期时间，则按“证书创建时间 + 730 天（365*2）”计算展示。

## 7. 管理员专属功能

### 7.1 用户与权限
在“用户与权限”页面可执行：
- 调整用户角色
- 手动邮箱验证
- 重置用户密码
- 编辑用户姓名/邮箱/角色
- 删除用户（谨慎）

### 7.2 系统设置
在“系统设置”页面可配置：
- SMTP 邮件服务（主机、端口、账号、密码、发件人）
- 通知规则（研发提醒、服务通知、过期预警）
- 关键人员邮箱标签列表

## 8. 常见问题（FAQ）

### Q1: 页面提示 `Failed to fetch` / 数据加载失败
可能原因：
- 服务未启动或端口不可达；
- 浏览器访问了错误环境（如本地 localhost 与线上混淆）；
- 当前版本存在后端异常。
建议：
1. 确认 URL 与环境；
2. 检查服务健康接口 `/api/health`；
3. 联系运维查看容器日志。

### Q2: 为什么我看不到某些工单？
系统按角色和账号做数据范围限制。  
若你需要查看他人数据，请联系管理员分配或确认权限策略。

### Q3: 已签发但无法下载
通常是证书文件缺失或路径不可用。请联系签发人员重新上传证书包。

## 9. 安全建议
- 首次登录后请立即修改密码。
- 不要在公共设备保存登录状态。
- 证书 ZIP 与解压密码属于敏感信息，请勿外传。
